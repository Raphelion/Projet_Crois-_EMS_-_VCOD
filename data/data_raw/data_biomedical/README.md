-- Données biomédicales --

Les données biomédicales sont des fichiers .txt que l'on a transformé en fichiers .csv avec les colonnes "nam", contenant probablement un identifiant de l'individu. 

Une colonne "charge_virale" fait référence à la quantité de virus présente dans un échantillon biologique d'un patient. C'est un indicateur souvent utilisé dans les traitements contre le VIH. C'est un chiffre entre 0 et 1, donc probablement un pourcentage ? Ou alors c'est une unité différente de quantité.

Une colonne "CD4" fait référence au nombre de lymphocytes dans le sang d'un patient. Une diminution du nombre de CD4 signifie que le système immunitaire est affaibli. Dans le VIH, le virus détruit les cellules, donc le suivi du CD4 permet de mesurer la progression de la maladie.

Une colonne "anticorps" fait référence au nombre d'anticorps présents dans le sang d'un patient. Moins il y a d'anticorps, plus le patient est vulnérable et faible, et plus le patient a d'anticorps, plus il a été exposé ou vacciné, donc il a une certaine immunité.

Ces données sont très intéressantes à exploiter, surtout quand on sait que le VIH est une maladie qui affecte de manière destructrice le système immunitaire.
